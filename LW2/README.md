# LW2

Task 2 folder for dog-image preprocessing and augmentation.

## Files

- `task.pdf` - assignment description
- `task-py.py` - Python implementation
- `task-matlab.m` - MATLAB implementation
- `dataset/` - input image dataset
- `results/` - generated outputs (created by Python script)

## Python Run

From `LW2/`:

```bash
python task-py.py
```

The script reads from `dataset/` and writes processed images to `results/`.
