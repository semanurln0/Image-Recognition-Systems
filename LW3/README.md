# LW3

Task 3 folder for text-region extraction and segmentation from an image.

## Files

- `task.pdf` - assignment description
- `task.py` - Python implementation
- `image.png` - input image
- `output/` - generated intermediate and final outputs

## Python Run

From `LW3/`:

```bash
python task.py
```

The script writes generated images and `threshold_report.json` into `output/`.
