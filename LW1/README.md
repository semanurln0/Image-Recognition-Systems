# LW1

Task 1 folder for basic image-processing workflow.

## Files

- `task.pdf` - assignment description
- `task-py.ipynb` - Python notebook solution
- `task-matlab.m` - MATLAB solution
- `image.jpg` - sample/input image
- `result.png` - generated result image

## How to Use

1. Review `task.pdf`.
2. Run either:
   - `task-py.ipynb` in Jupyter, or
   - `task-matlab.m` in MATLAB.
