# Image Recognition Systems Labworks

This repository contains lab tasks for the Image Recognition Systems course.

## Structure

- `LW1/` - Task 1 materials (`task-py.ipynb`, `task-matlab.m`, `task.pdf`)
- `LW2/` - Task 2 materials (`task-py.py`, `task-matlab.m`, `task.pdf`, dataset)
- `LW3/` - Task 3 materials (`task.py`, `task.pdf`, output images/reports)

## Quick Start

1. Open the task folder you want to run.
2. Read the related `task.pdf`.
3. Run the Python or MATLAB implementation for that task.

See each task folder README for task-specific notes.
